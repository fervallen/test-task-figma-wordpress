### Test Task Figma -> Wordpress theme
Used **Tailwind** for adaptiveness and **GSAP** for animations

#### Launching
1. Start docker service (requires docker-compose)  
`cd docker
docker-compose up -d`  
8080 port should be free to map  

2. Open http://localhost:8080 and go through basic WP installation process  
3. Log in into fresh WP admin panel -> Themes -> Mounty theme -> Activate  
4. Log out and visit http://localhost:8080 once more
